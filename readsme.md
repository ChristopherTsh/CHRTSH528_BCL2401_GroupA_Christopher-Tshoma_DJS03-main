# My Awesome Book Preview App

Welcome to my book preview app! This project was created to showcase my skills in JavaScript, HTML, and CSS. Below you'll find some important information about the project and how to get it up and running on your machine. I hope you enjoy it!

## Introduction

This project is a simple web application that displays a list of books. You can search for books by title, author, or genre, and you can also toggle between light and dark themes. It's a great example of how to work with JavaScript DOM manipulation and event handling.

## Features

- Display a list of books with their images, titles, and authors.
- Search for books by title, author, or genre.
- Toggle between light and dark themes.
- Responsive design that works on all devices.

## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

Make sure you have the following installed:

- A modern web browser (like Chrome, Firefox, or Edge)
- A code editor (like VS Code)


## Usage

Once the app is open in your browser, you can:

- Browse through the list of books.
- Use the search bar to filter books by title, author, or genre.
- Click on a book to see more details about it.
- Toggle between light and dark themes using the settings.
